ReadMe for MinecraftServerSync v${project.version}
--------------------

=== Installation ===
1. Have Java 8+ 64bit installed
   1.1 Check for 64bit by entering the command "java -d64 -version" in your command line
2. Forward/open port 25565 in your firewall

====== Usage =======
1. Read and edit config.properties
2. Launch MinecraftServerSync by executing start.bat
3. Start Server/Minecraft as necessary. The GUI should be self-explanatory.
